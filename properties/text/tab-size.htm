tab-size

  • 版本：CSS3
  • 媒体：视觉

语法：

tab-size：<integer> | <length>

默认值：8

适用于：块级元素

继承性：有

动画性：当值为 <length> 时

计算值：指定值

取值：

<integer>：
用整数值指定制表符的长度。不允许负值。
<length>：
用长度值指定制表符的长度。不允许负值。

说明：

检索或设置对象中的制表符的长度。
  • 该属性决定了制表符(U+0009)的宽度，整数代表空格(U+0020)的倍数(如：tab-size:4; 表示制表符宽度是4个空格的宽度)，不允许负数
  • 对应的脚本特性为tabSize。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6.0-10.0 4.0-18.0 #1 5.1.7 1.0-20.0 10.6-14.0 #1-o-
版本 21.0-31.0 #1 15.0-18.0 #1
  1. 不支持：<length>作为参数值。
