Media Queries: device-width

  • 版本：CSS3
  • 是否接受min/max：是

语法：

device-width：<length>

取值：

<length>：
用长度值来定义宽度。不允许负值

说明：

定义输出设备的屏幕可见宽度。
  • 本特性接受min和max前缀，因此可以派生出min-device-width和max-device-width两个媒体特性。

  • 简单列举几个应用示例：

    @media screen and (device-width:1024px){ … }
@import url(example.css) screen and (min-device-width:800px);
<link media="screen and (min-device-width:300px) and (max-device-width:900px)" rel="stylesheet" href="example.css" />

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
  • 橙色 = 通过第三方工具支持
