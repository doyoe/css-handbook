CSS Only Webkit Properties

  • Only Webkit属性

常用Webkit私有属性

属性 版本 继承性 描述
-webkit-text-fill-color CSS3 有 设置或检索对象中的文字填充颜色
-webkit-text-stroke CSS3 有 复合属性。设置或检索对象中的文字的描边
-webkit-text-stroke-width CSS3 有 设置或检索对象中的文字的描边厚度
-webkit-text-stroke-color CSS3 有 设置或检索对象中的文字的描边颜色
-webkit-box-reflect CSS3 有 设置或检索对象的倒影
-webkit-tap-highlight-color CSS3 无 设置或检索对象轻按时高亮
-webkit-user-drag CSS3 无 用以设置一个元素和它的内容是否可以被拖拽
-webkit-overflow-scrolling CSS3 无 指定元素是否使用native-style（滚动回弹效果）来滚动溢出内容

紫色 链接表示该属性在CSS3有修改或者增加了新的属性值；橙色 链接表示该属性是CSS3新增属性
