initial

  • 版本：CSS3

说明：

属性初始值。
  • initial属于 CSS-wide 关键字，这表示所有的属性都可以接受该值。
  • 如果你想重置某个属性为 User Agent 默认设置，那么就可以使用initial
  • 正常工作的例子：

    示例代码：

    ul { list-style: none; } .news-list { list-style: initial; }

    如上述代码，有可能ul会被reset掉list-style，如果你想某个地方需要默认定义，那么就可以直接设置为 initial

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
