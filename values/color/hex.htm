hex

  • 版本：CSS1

说明：

#RRGGBB 或 #RGB

取值：

RR：
红色值。十六进制正整数
GG：
绿色值。十六进制正整数
BB：
蓝色值。十六进制正整数

说明：

十六进制记法
  • 以上三个参数，取值范围为：00 - FF。
  • 参数必须是两位数。对于只有一位的，应在前面补零。
  • 如果每个参数各自在两位上的数字都相同，那么本单位也可缩写为 #RGB 的方式。例如：#FF8800 可以缩写为 #F80。
  • 此色彩模式与RGB色彩模式不同。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
