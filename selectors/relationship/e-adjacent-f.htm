相邻选择符(E+F)

  • 版本：CSS2

语法：

E+F { sRules }

说明：

选择紧贴在E元素之后F元素。
  • 与 兄弟选择符(E~F) 不同的是，相邻选择符只会命中符合条件的相邻的兄弟元素。

    E~F VS. E+F：

    <style> /* 相邻选择符(E+F) */ h3 + p { color: #f00; } /* 兄弟选择符(E~F) */ h3 ~ p { color: #f00; } </style> <h3>这是一个标题</h3> <p>p1</p> <p>p2</p> <p>p3</p>

    这个例子中，如果是相邻选择符，那么只有 p1 会变成红色；如果是兄弟选择符，那么 p1/p2/p3 都会变成红色；

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
