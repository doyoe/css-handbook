E:active

  • 版本：CSS1/2

语法：

E:active{ sRules }

说明：

设置元素在被用户激活（在鼠标点击与释放之间发生的事件）时的样式。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
